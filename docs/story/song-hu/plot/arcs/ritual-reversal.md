# 仪式反转弧

## Setup

第一关把“自下而上：杉木、白毛、蓬草”呈现为正确、可信、可学习的归位仪式顺序。玩家相信自己是在送狐神回到本社位置，让村庄恢复平静。

## Development

第二关让材料进入空间逻辑：杉木承托道路，铃绳转向石狐，狐火改变路径。玩家开始相信这些规则确实能作用于世界，同时也发现火越强、秩序越稳，白狐越像被固定住。

## Payoff

第三关揭示“迎狐”是“送狐”，“白毛”是“白衣”，“狐火”是送行火，“狐神归位”其实是把活人祭祀重新固定为狐神之位。玩家发现自己熟练执行的不是单纯修复，而是重启牺牲。
