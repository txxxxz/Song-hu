# Plot Structure

## Story Structure

**Model:** Three-Act Structure, mapped to three playable levels.

| Act | Level | Function | Player Belief |

| Act II | 断参道 | 让选择影响狐神之位稳定度 | 我可能在用姐姐的人性换取稳定 |
