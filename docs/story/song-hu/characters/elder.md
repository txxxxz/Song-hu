# 老人

## Function

老人是仪式权威的声音。她不需要像反派一样威胁玩家，而是用平静、重复、理所当然的语气把玩家带入“照做就对了”的状态。

## Voice

少解释，多强调顺序、规矩和先例。她的台词应让人感觉这不是第一次发生，也不会是最后一次发生。

## Key Line

“这两样都能成理，只是性子不同。绳子偏安抚，会把壶轻轻地请出来；火石会让火先行，再让壶现形，更偏催逼。”
