# 姐妹记忆弧

## Setup

主角从小被告知姐姐被白狐带走。这个事实一开始只作为背景阴影存在，不主动解释；玩家只知道姐姐的名字在家中和神社里越来越少被提起。

## Development

白狐的回望、停顿、脚印、人形影子、被刮掉一半的身高刻痕与档案里的半个名字，逐步把“神使”拉回“人”的方向。白狐既像在等主角认出她，也像在阻止主角说出口。

## Payoff

档案残片揭示纱夜身份与名字规则。最终选择“呼其名”意味着承认她是姐姐，而不是继续把她留在白狐这个安全称呼里；同时也意味着让她不再承担狐神之位，村庄平静随之松动。
