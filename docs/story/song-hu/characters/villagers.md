# 村人

## Function

村人不是单个反派，而是维持仪式叙事的共同体。老人和少数社守知道更多，普通村民多半只继承了恐惧、禁忌和“狐神必须归位”的说法，孩子、病人和后来出生的人并不是祭祀共犯。

## Moral Weight

村庄端必须有无辜者。玩家呼名后承受混乱的，不应只有隐瞒真相的老人，也包括并不知道真相却依赖平静生活的人。这样最终选择才是电车难题，而不是单纯惩罚村庄。

## Ending Use

结局 A 中，村人上山行礼并称赞仪式完成，体现秩序恢复，也体现他们再次无知地接受了牺牲。结局 B 中，村人陷入混乱，旧年失踪与火灾记忆重新浮现；同时，孩子惊醒、病人复发、山路失序，让玩家看到代价落在无辜者身上。
