# 纱夜

## Function

纱夜是故事的真实中心。她不是怪物，也不是单纯的神使，而是被仪式、文字和共同体记忆一起改写，并被固定在狐神之位上的人。

## Reveal Path

1. 白狐在第一关结尾回望玩家，像确认又像迟疑。
2. 第二关出现人形影子、焦黑小脚印、异常狐火。
3. 第三关档案残片拼出“本次送狐者，社守之女——纱夜”，并说明名字被记起会让狐神之位松动。
4. 最终石阶前，白狐回头，纸门影子变成穿白衣的小女孩。

## Dramatic Rule

在终局之前，不要直接把纱夜作为“姐姐”说破。她应该先作为停顿、回望、旧字、脚印、半个名字存在。她的回头也不能只像求救，还要带一点阻止：她想被记起，也知道自己一旦被叫名，村庄会重新承受混乱。
