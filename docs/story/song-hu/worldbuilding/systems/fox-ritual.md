# 迎狐之仪 / 送狐之仪

## Public Version

灾年或狐火不归时，村中需要把狐神送回本社之位。供物必须按次第奉上，否则狐神认不得路，村庄的混乱会继续扩大。

## True Version

仪式原名“送狐之仪”。村庄选出孩子作为送狐者，穿白衣、带狐火走入山里，通过仪式被固定为狐神之位，用自身存在压住灾厄、迷路、火灾、疫病和混乱记忆。后人把“送”刮改为“迎”，把“白衣”误写成“白毛”，把人的名字拆碎抹去，只留下狐神。

## Core Reversal

玩家以为自己在送狐神归位，其实是在把一个曾经的人重新固定回狐神之位。最终呼名会让她恢复人的身份，同时让被镇住的混乱回到村庄。
