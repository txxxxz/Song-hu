# 狐神之位

## Public Version

狐神是守护村庄的山中神使。狐火不归时，狐神离位，村庄会出现山路错乱、灯火熄灭、病痛复发和灾厄回潮。

## True Version

狐神之位不是天然神明，而是由活人祭祀维持的镇灾位置。被选中的人会被仪式从“人”改写成“狐神”，以自己的名字、记忆和人生作为代价，压住村庄的混乱。

## Name Rule

不能有人完整记得并叫出送狐者的真名。名字是她作为人的锚点；一旦被最亲近的人叫出口，狐神之位会松动，她会恢复人的身份，被镇住的混乱也会回到村庄。

## Dramatic Use

这个设定让最终选择成为电车难题：呼名不是无代价救人，不呼名也不是单纯懦弱。玩家在一个无辜者的人格与许多无辜者的平静之间做选择。
