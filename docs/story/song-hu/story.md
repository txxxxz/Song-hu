# 送狐

## Synopsis

《送狐》发生在一座被群山包围、供奉稻荷社的旧村落。玩家扮演神社见习巫女，在狐火不归、山路错乱、村中旧病与恐慌重新浮现的夜晚，被要求完成“送狐神归位”的古老仪式。

玩家通过御供筒的栈式供物系统学习并执行仪式，沿着装束之祠、断参道、本社一路追随白狐。最终她拼出真相：所谓狐神不是外来的神明，而是被活人祭祀固定成的“狐神之位”；那只白狐正是多年前被送上山、以名字和人生作为代价镇住村庄混乱的姐姐纱夜。

## Core Story Promise

玩家最初以为自己在修复神社、送狐神归位；中段开始怀疑仪式的异常；终局理解所谓归位，其实是把一个被献祭的人重新固定回狐神位置。最终选择不是简单的善恶判断，而是在“叫出姐姐的名字，让她重新作为人被承认”与“维持村中许多无辜者赖以生活的平静”之间承担代价。

## Tone & Style

基调是克制、潮湿、冷静的民俗悬疑。恐怖不靠怪物追逐，而靠规矩的熟悉感、环境里的轻微不协调、被刮改的文字、重复童谣和玩家事后重读自己行为时产生的后劲。叙事语气应避免直接解释，优先让物件、顺序、火光、影子和残损记录承担信息。

## World Premise

旧村相信山路、田地收成、火灾与疫病都受“狐神是否归位”影响。狐神离位时，山路会反复绕回、灯火无故熄灭、旧病复发、村人的记忆出现空洞，连无辜的孩子和病人也会被混乱卷入。灾年或狐火不归时，村中举行“迎狐之仪”，公开说法是送狐神回到本社，让秩序恢复。

真实历史中，这套仪式原名“送狐之仪”。灾年里，村庄会选出一个孩子作为“送狐者”，穿上白衣、带着狐火走进山里，最终被仪式固定为狐神之位，用自身存在压住山火、疫病、迷路和共同体记忆中的混乱。代价是不能再有人完整记得她作为人的名字；一旦真名被最亲近的人叫出，狐神之位会松动，她会重新恢复人的身份，但村庄被压住的混乱也会回到世界。

后人为了让这套秩序能继续被接受，把“送狐”改写成“迎狐”，把“白衣”误写成“白毛”，把人的名字拆成残片，只留下狐神与仪式。普通村民多半只继承了恐惧和规矩，并不完整知道祭祀真相；真正压迫玩家选择的，不是惩罚坏人，而是许多无辜者也依赖这份平静活下去。

## Player Character

玩家角色是神社见习巫女，自幼在山社附近长大，对童谣、祭具和禁忌并不陌生，但从未进入仪式最深处。她小时候被告知姐姐纱夜是在某年夜里被白狐带走，直到终局才意识到白狐一直不是神使，而是被送走后仍在狐神之位上承受孤独的姐姐。她的选择会决定纱夜是否被重新作为人承认，也会决定村庄是否继续维持表面平静。

## Narrative Pillars

- 玩法即仪式：御供筒的后进先出、供物顺序和祭坛读取方向，是玩家对“归位”秩序的身体学习。
- 仪式即固定：杉木、白衣、蓬草、铃绳、狐火都同时是谜题道具、民俗规则和把人固定成狐神的真相证据。
- 名字即破封：最终“呼其名”不是战斗命令，而是承认狐神曾经是人，并让狐神之位松动。
- 两端都要无辜：纱夜是无辜牺牲者，村中下一代和弱者也并非祭祀共犯。
- 两个结局都要有代价：沉默带来稳定但完成第二次牺牲；呼名带来救赎但让村庄重新承受混乱。

## Notes

后续新增角色、地点、系统、关卡文本时，应优先检查它们是否服务四个核心点：`迎狐 -> 送狐`、`白毛 -> 白衣`、`白狐 -> 纱夜`、`狐神归位 -> 活人重新被固定为狐神之位`。最终选择必须始终保持为电车难题，而不是单纯揭穿恶俗。
