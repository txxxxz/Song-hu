# 玩家服从到怀疑弧

## Setup

玩家从老人和祭坛处获得清晰规则，先学会把事情做对，并相信这些规则会救下村庄。

## Development

玩家发现即使规则正确，世界也出现不协调：火不自然、影子不对、童谣越听越不像禁忌而像警告。更重要的是，玩家发现自己的选择会让白狐更稳定，也更孤独。

## Payoff

最终选择不再是“哪种供物更有效”，而是“是否继续服从一个靠不认人来成立的秩序”。玩家必须承担自己对纱夜和村庄两边造成的后果。
