# 第一关：装束之祠

## Dramatic Goal

让玩家相信送狐神归位是必要且正确的，并通过御供筒学习“反向收集、正向释放”的供物顺序。

## Key Beats

- 开场先给出村庄混乱的压力：山路错乱、山下灯火熄灭、远处孩子哭声或病人咳声。
- 老人在祠前交给玩家御供筒，强调顺序不能乱，狐神必须归位。
- 玩家收集蓬草、白色纤维状材料、杉木，并在祭坛形成“杉木 -> 白毛 -> 蓬草”。
- 童谣第一次自然出现。
- 顶礼供物放在长桌上：铃绳纤维与狐火石。
- 玩家选择安抚或催逼路径，白狐出现并上山。安抚让它更像迟疑的人，催逼让道路更快稳定但火光更强。

## Must Foreshadow

白狐的停顿和回望必须像“认人”，但此时不能解释。村庄端的无辜压力也要在第一关埋下，否则最终选择会失衡。
