# 白狐

## Function

白狐是被改名后的纱夜，也是玩家对真相的误读界面。它在前两关承担引路者形象，第三关被重新解释为“认人”和“阻止被认出”同时存在的矛盾姿态。

## Behavior Rules

- 安抚路线：停在关键节点，稳定回望，不逼近。
- 强化路线：狐火更亮，影子更容易露出人形，脚印更焦黑。
- 终局：真正回头，触发玩家是否呼名的选择。它不能只表现成等待救援，也要显出“如果你叫我，村庄会乱”的迟疑。
