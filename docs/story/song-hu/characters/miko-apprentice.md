# 见习巫女

## Function

玩家操控的主角。她熟悉神社边缘的规矩，却没有真正参与过仪式深处，因此可以自然承担教学视角。

## Inner Arc

她从“照规矩做才能救村子”，走向“规矩本身可能是在把一个人重新固定成狐神”。最终选择沉默或呼名，决定她是否继续让姐姐被称作白狐，也决定村庄是否继续拥有靠牺牲换来的表面平静。

## Key Knowledge

- 知道童谣“狐上山，不回头；若回头，就莫再唤其名”。
- 小时候被告知姐姐被白狐带走。
- 逐步发现村中混乱会伤及无辜者，不只是惩罚旧制度。
- 直到主社档案间才知道纱夜是“本次送狐者”，也是被固定在狐神之位上的人。
