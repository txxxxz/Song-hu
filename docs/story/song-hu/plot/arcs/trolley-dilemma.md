# 电车难题弧

## Setup

第一关开场要让玩家看到村庄混乱不是抽象背景：山下灯火乱、有人迷路、病人旧症复发、孩子被吓醒。玩家相信送狐神归位是为了保护很多无辜的人。

## Development

第二关逐步揭示仪式稳定度与白狐痛苦相关。安抚选择让白狐更像人、更迟疑；催逼选择让道路更稳定、狐火更强，但白狐更像被压回某个位置。

## Climax

第三关拼出真相：纱夜是狐神之位的承担者，叫出她的名字会让她恢复人的身份，也会解除村庄平静的支撑。

## Resolution

结局 A：玩家沉默，村庄恢复表面平静，纱夜继续孤独地作为狐神。  
结局 B：玩家呼名，纱夜重新被承认为人，村庄混乱回潮，无辜者也承受代价。

## Design Rule

不要把任何一边写成纯正确。呼名不是完美善举，沉默也不是单纯懦弱；二者都必须留下刺。
