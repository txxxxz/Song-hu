# 《送狐》全台本与木牌设置

## 写法约束

- 老人只在 `L1-01` 出现。任务交完后隐藏，不再远程说话，也不在结局出现。
- 文本采用日式 RPG 的节奏：场景短，台词自然，记录可读。玩家不用停下来读长论文。
- 关键路径约 15 分钟可结束。每个场景只承担一件新信息。
- 真相不直接用一句话报给玩家。白色供物、仪式名、送行者身份和最终代价都通过两处以上线索互相补齐。
- 第三幕允许主角做少量整理，但不替玩家讲完答案。
