# 第三关：本社·不返之灯

## Dramatic Goal

降低操作复杂度，提高信息拼接密度，完成“迎狐 -> 送狐”、“白毛 -> 白衣”、“白狐 -> 纱夜”、“狐神归位 -> 活人被固定成狐神之位”的连续反转。

## Key Beats

- 主社完整但不自然，像被长期维持。
- 狐火照出旧木额下方被刮掉的“送狐”字迹。
- 档案碎片揭示灾年送行、供物真实含义、狐火用途和狐神之位规则。
- 名册残片、后墙身高刻痕与童谣残页共同拼出“本次送狐者，社守之女——纱夜”。
- 最终石阶前白狐回头，影子变为穿白衣的小女孩；她既像求救，也像阻止。
- 玩家选择沉默或呼名，并在选择前清楚知道两边都有无辜代价。
