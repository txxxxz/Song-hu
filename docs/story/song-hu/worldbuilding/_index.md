# Worldbuilding

## World Overview

世界由山、旧村、稻荷社和一套被改名的灾年仪式构成。村人相信狐神归位才能让山路、收成、疫病与火灾恢复秩序；真实机制是共同体把活人祭祀包装为神位回归，并通过一代代仪式文本让人的名字和代价失真。
